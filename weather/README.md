
pyweather
===================================
  				  ||
    Linux 命令行天气查询小工具 	  ||
			          ||
====================================

安装说明：


使用参数：
	pyweather 	  (没有参数，根据设定的默认值查询) 
	pyweather  -a     (-auto 联网后根据IP所在地，自动获取本地天气)
	pyweather  -s   [cityname]  (-set设定默认查询天气的城市)
	pyweather  -c   [cityname]  (-city 查询指定城市的天气)
	pyweather  -A   (所有城市的天气)
